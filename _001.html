Simplicity Matters

In C: 

#include   /* printf */
#include  /* size_t */
#include  /* EXIT_SUCCESS */

int main (void)
{
    size_t i;
    int    array[] = {1, 1, 2, 3, 5, 8, 13, 21};

    for (i=0; i<(sizeof array/sizeof(int)); i++)
    {
        printf("The next number is: %2d\n", array[i]);
    }
    
    return EXIT_SUCCESS;
}

In Python: 

for number in (1, 1, 2, 3, 5, 8, 13, 21):
    print("The next number is: {:>2}".format(number))

Both has the same output:

The next number is:  1
The next number is:  1
The next number is:  2
The next number is:  3
The next number is:  5
The next number is:  8
The next number is: 13
The next number is: 21
